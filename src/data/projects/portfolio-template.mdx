# Portfolio Template

## Overview

A modern, customizable portfolio template designed for developers and designers to showcase their work with style.

## Features

- Dark/Light mode with smooth transitions
- Beautiful page animations
- Responsive design
- SEO optimized
- Fast page loads
- Easy customization

## Technical Highlights

### Animation System

The template uses Framer Motion for:

- Page transitions
- Scroll-based animations
- Hover effects
- Loading states

### Theme System

Implemented a robust theme system:

- CSS variables for easy customization
- System preference detection
- Smooth theme transitions
- Persistent user preference

### Performance

Optimized for Core Web Vitals:

- Image optimization with Next.js Image
- Font optimization
- Code splitting
- Lazy loading

## Customization

The template is designed to be easily customizable:

- Edit content via markdown/MDX
- Modify colors in one config file
- Add/remove sections easily
- Custom components library

## Getting Started

Clone the repository and customize for your needs. The template includes detailed documentation for customization.
